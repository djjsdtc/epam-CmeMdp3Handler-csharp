# epam-CmeMdp3Handler-csharp

EPAM's CME MDP 3.0 Java Handler, porting to .NET 8+ (C#)

Original project: https://github.com/epam/java-cme-mdp3-handler

This project contains the package code and sample programs in original README file.

Differences with original Java project are listed in JAVA_TO_CSHARP_DIFFERENCES.md



## Package dependencies

- Microsoft.Extensions.Logging (8.0.0)

- Microsoft.Extensions.Logging.Console (8.0.0)
